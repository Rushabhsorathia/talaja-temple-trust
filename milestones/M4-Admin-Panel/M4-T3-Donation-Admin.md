# M4-T3 — Donation Management & Reconciliation

- **Type:** Story
- **Priority:** High
- **Story Points:** 5
- **Milestone:** M4 – Admin Panel
- **Stack:** Filament resources

## User Story
> As an officer, I want to manage donations, verify offline payments, and reconcile with gateway.

## Acceptance Criteria
- [ ] Donation list with filters (date, category, mode, status).
- [ ] Manual entry for offline (cash/cheque/UPI) donations.
- [ ] Receipt re-issue/void.
- [ ] 80G toggle + bulk 80G generation.
- [ ] Gateway reconciliation report.
- [ ] Export CSV/PDF.

## Definition of Done
Offline + online donations tracked; reconciliation report matches gateway.
