# M4-T5 — Financial Management (Receipts, Payments, Bank Reconciliation)

- **Type:** Story
- **Priority:** High
- **Story Points:** 8
- **Milestone:** M4 – Admin Panel
- **Stack:** Filament

## User Story
> As a finance officer, I want to record receipts/payments and reconcile bank statements.

## Acceptance Criteria
- [ ] Receipt management (auto + manual).
- [ ] Payment management (vendors, salaries, expenses).
- [ ] Ledger / day book / cash book views.
- [ ] Bank statement CSV import + auto + manual match.
- [ ] Reconciliation status per transaction.
- [ ] P&L + balance sheet summary view.
- [ ] Approval workflow for payments > threshold.

## Definition of Done
Books reconcile to bank; reports exportable.
