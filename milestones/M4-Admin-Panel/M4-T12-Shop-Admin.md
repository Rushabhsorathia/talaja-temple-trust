# M4-T12 — Shop / Order Fulfilment Admin

- **Type:** Story
- **Priority:** Medium
- **Story Points:** 5
- **Milestone:** M4 – Admin Panel
- **Stack:** Filament

## User Story
> As staff, I want to manage products and fulfil orders.

## Acceptance Criteria
- [ ] Product CRUD (image, price, stock, EN/GU).
- [ ] Order list with status (new/packed/shipped/delivered/cancelled).
- [ ] Invoice printing.
- [ ] Inventory management + low-stock alerts.
- [ ] Shipping/awb capture.

## Definition of Done
Order lifecycle manageable end-to-end.
