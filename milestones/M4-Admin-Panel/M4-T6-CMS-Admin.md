# M4-T6 — CMS Admin (Content, Banners, Media)

- **Type:** Story
- **Priority:** High
- **Story Points:** 5
- **Milestone:** M4 – Admin Panel
- **Stack:** Filament + Spatie Media Library

## User Story
> As an admin, I want to manage all public content without developer help.

## Acceptance Criteria
- [ ] Resources: pages, banners (scheduled), news, galleries, videos, publications, downloads, FAQs, homepage builder, contact.
- [ ] Rich text editor with EN/GU.
- [ ] Media library (S3) with image optimization.
- [ ] Publish/unpublish, scheduling, version history.
- [ ] SEO fields per item.

## Definition of Done
Content changes reflect on public site live.
