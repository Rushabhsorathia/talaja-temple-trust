# M4-T8 — Notice & Communication Manager (SMS/Email/WhatsApp)

- **Type:** Story
- **Priority:** High
- **Story Points:** 5
- **Milestone:** M4 – Admin Panel
- **Stack:** Laravel Queue + Filament

## User Story
> As an admin, I want to send notices/notifications via SMS, email, and WhatsApp to devotees.

## Acceptance Criteria
- [ ] Template manager (SMS/email/WhatsApp) with EN/GU + variables.
- [ ] Audience selector (all, donors, guests, festival-based).
- [ ] Bulk send via queue (Redis).
- [ ] Schedule send.
- [ ] Delivery logs + retry.
- [ ] Auto-trigger rules (donation success, booking reminder).

## Definition of Done
Bulk send completes with delivery logs.
