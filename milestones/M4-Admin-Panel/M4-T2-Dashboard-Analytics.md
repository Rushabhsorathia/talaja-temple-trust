# M4-T2 — Role-Based Dashboard & Analytics

- **Type:** Story
- **Priority:** Highest
- **Story Points:** 8
- **Milestone:** M4 – Admin Panel
- **Stack:** Filament widgets + Vue charts (ApexCharts)

## User Story
> As a trustee, I want a personalized dashboard with KPIs and revenue trends.

## Acceptance Criteria
- [ ] Daily summary tiles: donations, visitors, bookings, events, financials.
- [ ] Temple-wise statistics (if multi-temple).
- [ ] Revenue dashboard with drill-down (day/week/month/year).
- [ ] Graphical MIS: bar/line/pie (ApexCharts).
- [ ] Role-specific widget visibility.
- [ ] Date-range filters.
- [ ] Export dashboard data.

## Definition of Done
Dashboard loads < 2s with real data.
