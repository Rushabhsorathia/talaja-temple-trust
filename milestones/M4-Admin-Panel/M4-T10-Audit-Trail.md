# M4-T10 — Audit Trail (Activity, Transaction, Config)

- **Type:** Story
- **Priority:** High
- **Story Points:** 5
- **Milestone:** M4 – Admin Panel
- **Stack:** spatie/laravel-activitylog + Filament

## User Story
> As an auditor, I want complete audit trails of all admin actions.

## Acceptance Criteria
- [ ] User activity log (page, action, IP, timestamp).
- [ ] Transaction audit (immutable log table).
- [ ] Configuration audit (who changed what, diff).
- [ ] Filter by user, date, module.
- [ ] Export to PDF/CSV.
- [ ] Tamper-evident storage.

## Definition of Done
Any change traceable to a user + timestamp.
