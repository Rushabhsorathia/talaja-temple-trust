# M4-T4 — Accommodation Operations (Check-in/out, Housekeeping)

- **Type:** Story
- **Priority:** High
- **Story Points:** 8
- **Milestone:** M4 – Admin Panel
- **Stack:** Filament + Vue calendar

## User Story
> As staff, I want to manage room inventory, check-ins/outs and housekeeping.

## Acceptance Criteria
- [ ] Room & hall inventory CRUD (type, tariff, amenities).
- [ ] Booking calendar (overview, drag to assign).
- [ ] Check-in / check-out workflow with ID capture.
- [ ] Housekeeping status (dirty/clean/inspected) auto + manual.
- [ ] Walk-in booking support.
- [ ] Occupancy report.
- [ ] Extend/stay modification.

## Definition of Done
Front-desk can run a full guest cycle.
