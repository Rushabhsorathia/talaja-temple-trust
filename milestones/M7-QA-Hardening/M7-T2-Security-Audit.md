# M7-T2 — Security Audit & Hardening

- **Type:** Story
- **Priority:** Highest
- **Story Points:** 5
- **Milestone:** M7 – QA, Hardening & Backup

## User Story
> As an admin, I want a hardened, secure application.

## Acceptance Criteria
- [ ] OWASP top-10 review.
- [ ] HSTS, CSP, X-Frame-Options, X-Content-Type headers.
- [ ] Rate limiting on auth/donate/OTP.
- [ ] SQL injection / XSS / CSRF checks.
- [ ] Secrets in vault, not in repo.
- [ ] Dependency vulnerability scan.
- [ ] Penetration test (3rd party or internal).

## Definition of Done
No critical/high findings open.
