# M7-T6 — Backup & Disaster Recovery

- **Type:** Story
- **Priority:** Highest
- **Story Points:** 5
- **Milestone:** M7 – QA, Hardening & Backup

## User Story
> As an admin, I want reliable backups so data is never lost.

## Acceptance Criteria
- [ ] Daily DB dump + media sync to S3 + local.
- [ ] Retention: 30 daily, 12 monthly.
- [ ] Encrypted backups.
- [ ] Restore drill documented + executed.
- [ ] Backup health monitoring/alerts.

## Definition of Done
Successful restore drill to a fresh environment.
