# M7-T4 — WCAG 2.1 AA Accessibility Audit

- **Type:** Story
- **Priority:** High
- **Story Points:** 5
- **Milestone:** M7 – QA, Hardening & Backup

## User Story
> As a user with disabilities, I want an accessible site.

## Acceptance Criteria
- [ ] Keyboard navigation, focus management.
- [ ] ARIA labels, alt text (EN/GU).
- [ ] Color contrast AA.
- [ ] Screen-reader test (NVDA/VoiceOver).
- [ ] axe-core automated scan in CI.
- [ ] Lighthouse Accessibility ≥ 95.

## Definition of Done
Accessibility audit passes with no critical issues.
