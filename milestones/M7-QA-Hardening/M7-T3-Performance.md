# M7-T3 — Performance & Caching Optimisation

- **Type:** Story
- **Priority:** High
- **Story Points:** 5
- **Milestone:** M7 – QA, Hardening & Backup

## User Story
> As a user, I want fast page loads.

## Acceptance Criteria
- [ ] Redis caching for CMS pages, dashboards.
- [ ] Image optimization + lazy load + WebP.
- [ ] CDN (Cloudflare) for static assets.
- [ ] DB query optimization + indexes.
- [ ] Queue heavy jobs.
- [ ] Lighthouse Performance ≥ 85.

## Definition of Done
Lighthouse ≥ 85 on key pages.
