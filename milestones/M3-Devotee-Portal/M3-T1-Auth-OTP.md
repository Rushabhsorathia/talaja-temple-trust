# M3-T1 — Devotee Register / Login (OTP + Password)

- **Type:** Story
- **Priority:** Highest
- **Story Points:** 5
- **Milestone:** M3 – Devotee Portal
- **Stack:** Laravel Sanctum + Vue

## User Story
> As a devotee, I want to register/login via mobile OTP or email+password.

## Acceptance Criteria
- [ ] Register with mobile/email.
- [ ] OTP via SMS (MSG91) + email.
- [ ] Password login + forgot/reset flow.
- [ ] Remember-me, session timeout.
- [ ] Account verification.
- [ ] Rate limiting on OTP requests.

## Definition of Done
Devotee can create account and log in securely.
