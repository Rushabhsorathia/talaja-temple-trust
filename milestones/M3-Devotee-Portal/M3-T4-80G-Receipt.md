# M3-T4 — 80G Receipt PDF Generation

- **Type:** Story
- **Priority:** High
- **Story Points:** 5
- **Milestone:** M3 – Devotee Portal
- **Stack:** Laravel (DomPDF/Snappy) + Vue

## User Story
> As a donor, I want a valid 80G tax-exemption receipt I can download.

## Acceptance Criteria
- [ ] 80G receipt template (trust name, reg no, PAN, donor PAN, amount, date).
- [ ] Sequential receipt numbering, unique code.
- [ ] Digital layout matches compliance format.
- [ ] Auto-generate on donation success (if eligible).
- [ ] Downloadable from "My Donations" + email attachment.
- [ ] Re-issue/void with audit log.

## Definition of Done
Generated PDF passes trust accountant review.
