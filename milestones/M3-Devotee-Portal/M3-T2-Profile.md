# M3-T2 — Devotee Profile & Account

- **Type:** Story
- **Priority:** High
- **Story Points:** 3
- **Milestone:** M3 – Devotee Portal
- **Stack:** Laravel + Vue

## User Story
> As a devotee, I want to manage my profile details.

## Acceptance Criteria
- [ ] Edit name, email, mobile, address (EN/GU).
- [ ] 80G profile fields (PAN, name-as-per-PAN).
- [ ] Change password.
- [ ] Download personal data (DPDP).
- [ ] Deactivate account.

## Definition of Done
Profile updates persist; validation enforced.
