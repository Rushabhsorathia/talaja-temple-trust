# M3-T8 — Shop: Cart, Checkout, Order History

- **Type:** Story
- **Priority:** Medium
- **Story Points:** 8
- **Milestone:** M3 – Devotee Portal
- **Stack:** Laravel + Vue + Razorpay

## User Story
> As a devotee, I want to buy trust publications/souvenirs/prasad online.

## Acceptance Criteria
- [ ] Product catalog (image, price, stock, EN/GU).
- [ ] Product detail page.
- [ ] Cart (add/remove/qty), persistent for logged-in users.
- [ ] Checkout (address, shipping, Razorpay).
- [ ] Order confirmation email+SMS.
- [ ] Order history + invoice PDF.
- [ ] Inventory decrement.
- [ ] Admin fulfilment status (M4).

## Definition of Done
Order placed, stock updated, invoice generated.
