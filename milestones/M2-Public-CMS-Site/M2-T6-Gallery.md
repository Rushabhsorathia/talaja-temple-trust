# M2-T6 — Photo & Video Gallery

- **Type:** Story
- **Priority:** Medium
- **Story Points:** 5
- **Milestone:** M2 – Public CMS Site
- **Stack:** Laravel + Vue (Inertia)

## User Story
> As a devotee, I want to browse photos and videos of temple events.

## Acceptance Criteria
- [ ] Photo gallery with lightbox, category filter, pagination.
- [ ] Video gallery (YouTube embeds) with category filter.
- [ ] Admin upload with image optimization (Intervention).
- [ ] Alt text EN/GU for accessibility.
- [ ] Lazy loading.

## Definition of Done
Galleries load fast; admin can add/remove media.
