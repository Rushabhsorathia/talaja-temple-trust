# M2-T3 — Homepage

- **Type:** Story
- **Priority:** Highest
- **Story Points:** 8
- **Milestone:** M2 – Public CMS Site
- **Stack:** Laravel + Vue (Inertia)

## User Story
> As a visitor, I want an inspiring homepage that showcases the temple, services and latest updates.

## Acceptance Criteria
- [ ] Hero slider / video background with mute toggle.
- [ ] Advisory banner (inline + modal).
- [ ] "About the Trust" teaser + Read More.
- [ ] "Online Services & Offerings" cards (Live Darshan, Donate, Booking, Shop).
- [ ] News & Updates grid (latest 6).
- [ ] Facilities & Offerings grid.
- [ ] Quick CTA cards (Visit, Gallery, History, Trustees).
- [ ] Live badge linking to Live Darshan.
- [ ] All sections CMS-driven (admin can reorder/hide).

## Tasks
- `Home.vue` with section components.
- Filament homepage builder (sections toggle + order).
- API endpoints `/api/home-data`.

## Definition of Done
Homepage matches approved hi-fi; content editable in admin.
