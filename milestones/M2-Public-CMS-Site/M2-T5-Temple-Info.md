# M2-T5 — Temple Info (Timings, Aarti, Pooja, Festivals, Map)

- **Type:** Story
- **Priority:** High
- **Story Points:** 5
- **Milestone:** M2 – Public CMS Site
- **Stack:** Laravel + Vue (Inertia)

## User Story
> As a devotee, I want temple timings, schedules, festival calendar and location so I can plan my visit.

## Acceptance Criteria
- [ ] Darshan timings table (CMS).
- [ ] Aarti schedule (name, time, EN/GU).
- [ ] Pooja schedule (name, time, fee).
- [ ] Festival calendar (month view, EN/GU).
- [ ] Google Maps embed + address.
- [ ] All editable via Filament.

## Definition of Done
Page renders CMS data; festival calendar filters by month.
