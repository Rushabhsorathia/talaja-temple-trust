# M2-T8 — Facilities & Offerings Grid

- **Type:** Story
- **Priority:** Medium
- **Story Points:** 5
- **Milestone:** M2 – Public CMS Site
- **Stack:** Laravel + Vue (Inertia)

## User Story
> As a devotee, I want to see facilities (Dharamshala, Vishram Gruh, Annashetra, Havan Khand, etc.) the trust offers.

## Acceptance Criteria
- [ ] Facilities grid (image, title, short desc, link).
- [ ] Detail page per facility (gallery, description, booking CTA where relevant).
- [ ] CMS-driven & translatable.
- [ ] Examples: Dharamshala, Rest House, Lapsi Ghar, Annashetra, Havan Khand, Medical, Environment, Cultural.

## Definition of Done
All facilities populated; detail pages render.
