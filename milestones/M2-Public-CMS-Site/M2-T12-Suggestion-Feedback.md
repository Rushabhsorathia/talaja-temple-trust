# M2-T12 — Public Suggestion & Feedback Form

- **Type:** Story
- **Priority:** Low
- **Story Points:** 3
- **Milestone:** M2 – Public CMS Site
- **Stack:** Laravel + Vue

## User Story
> As a devotee, I want to submit suggestions/feedback to the trust.

## Acceptance Criteria
- [ ] Public form (name optional, category, message, rating).
- [ ] Captcha + rate limiting.
- [ ] Stored in `feedbacks` table.
- [ ] Email notification to admin.
- [ ] Admin triage view (M4).

## Definition of Done
Form submissions persist & notify admins.
