# M2-T10 — Global & Advanced Search

- **Type:** Story
- **Priority:** Medium
- **Story Points:** 5
- **Milestone:** M2 – Public CMS Site
- **Stack:** Laravel (Scout/Meilisearch) + Vue

## User Story
> As a user, I want to search across news, pages, facilities, galleries, downloads.

## Acceptance Criteria
- [ ] Global search bar in header.
- [ ] Results grouped by type (news, page, facility, gallery).
- [ ] Advanced search with multi-parameter filters (date, type, category).
- [ ] Export results to CSV/PDF.
- [ ] Search analytics logged.

## Definition of Done
Search returns relevant results in < 500ms.
