# M2-T7 — News & Updates (List + Detail)

- **Type:** Story
- **Priority:** High
- **Story Points:** 5
- **Milestone:** M2 – Public CMS Site
- **Stack:** Laravel + Vue (Inertia)

## User Story
> As a devotee, I want to read the latest news and event updates from the trust.

## Acceptance Criteria
- [ ] News list page (cards: image, title, date, excerpt).
- [ ] News detail page (rich content, gallery, EN/GU).
- [ ] Publish/unpublish, scheduling.
- [ ] Category & tag filter.
- [ ] Share buttons (WhatsApp, Facebook).
- [ ] Featured news on homepage (via M2-T3).

## Definition of Done
CRUD in Filament; public pages paginated & filtered.
