# M2-T11 — SEO + CMS (Filament) for All Content

- **Type:** Story
- **Priority:** High
- **Story Points:** 5
- **Milestone:** M2 – Public CMS Site
- **Stack:** Laravel + Filament

## User Story
> As a Trust admin, I want to manage all public content and SEO from one panel.

## Acceptance Criteria
- [ ] Filament resources: pages, banners, news, gallery, videos, publications, downloads, FAQs, contact.
- [ ] Per-page SEO fields (meta title, description, OG image).
- [ ] Auto sitemap.xml + robots.txt.
- [ ] Breadcrumbs + canonical tags.
- [ ] Banner management with scheduling.
- [ ] Media library (S3-backed).

## Definition of Done
Admin can publish/translate/SEO-edit any page.
