# M1-T2 — Brand Identity & Design System

- **Type:** Story
- **Priority:** Highest
- **Story Points:** 5
- **Milestone:** M1 – Discovery & Design

## User Story
> As a Designer, I want a brand identity & design system so that all screens are visually consistent and culturally appropriate.

## Acceptance Criteria
- [ ] Color palette (saffron / deep red / cream / gold).
- [ ] Typography: serif headings (Cinzel / Tiro Devanagari Gujarati), sans body (Inter / Noto Sans).
- [ ] Logo treatment & favicon.
- [ ] Iconography set (temple, donation, booking, etc.).
- [ ] Design tokens (colors, spacing, radius, shadows) documented.
- [ ] Figma component library created.

## Definition of Done
Design tokens approved; Figma library published.
