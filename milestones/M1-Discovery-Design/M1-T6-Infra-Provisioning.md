# M1-T6 — Infrastructure & Sandbox Provisioning

- **Type:** Story
- **Priority:** High
- **Story Points:** 3
- **Milestone:** M1 – Discovery & Design

## User Story
> As a DevOps Engineer, I want infra & third-party sandboxes ready so development is unblocked.

## Acceptance Criteria
- [ ] Domain registered / DNS access confirmed.
- [ ] Hosting server (VPS/Cloud) provisioned (dev + staging).
- [ ] SSL (Let's Encrypt) + Cloudflare configured.
- [ ] S3-compatible storage bucket created.
- [ ] Razorpay sandbox account.
- [ ] MSG91/Twilio sandbox account.
- [ ] SendGrid/SES sandbox account.
- [ ] WhatsApp Business test number.

## Definition of Done
Credentials stored in vault; all sandboxes tested.
