# M1-T7 — Tech Stack Setup (Laravel + Vue)

- **Type:** Story
- **Priority:** High
- **Story Points:** 3
- **Milestone:** M1 – Discovery & Design

## User Story
> As a Lead Developer, I want the project scaffold ready so the team can start building features.

## Acceptance Criteria
- [ ] Laravel 11 + PHP 8.3 project initialized.
- [ ] Vue 3 (via Inertia.js or Vite SPA) integrated.
- [ ] Pinia for state, Vue Router (if SPA) configured.
- [ ] Tailwind CSS + design tokens wired.
- [ ] Filament PHP admin scaffolding.
- [ ] MySQL 8 + Redis connection verified.
- [ ] CI/CD pipeline (GitHub Actions) configured.
- [ ] Coding standards (Pint, ESLint, Prettier) enforced.
- [ ] README with setup steps.

## Definition of Done
New dev can `composer install && npm install && php artisan serve` and run app.
