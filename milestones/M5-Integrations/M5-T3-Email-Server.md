# M5-T3 — Email Server (SendGrid / SES)

- **Type:** Story
- **Priority:** High
- **Story Points:** 2
- **Milestone:** M5 – Integrations
- **Stack:** Laravel Mail + Markdown mailables

## User Story
> As a user, I want transactional emails (receipts, vouchers, OTP).

## Acceptance Criteria
- [ ] Branded HTML templates (EN/GU).
- [ ] Queue-based sending.
- [ ] Attachment support (PDFs).
- [ ] Bounce/complaint tracking.
- [ ] SPF/DKIM/DMARC configured.

## Definition of Done
Emails land in inbox with attachments.
