# M5-T7 — REST API + Sanctum Auth + Webhooks

- **Type:** Story
- **Priority:** High
- **Story Points:** 8
- **Milestone:** M5 – Integrations
- **Stack:** Laravel API + Sanctum

## User Story
> As an integrator, I want secure REST APIs and webhooks for third-party systems.

## Acceptance Criteria
- [ ] REST endpoints: donations, bookings, content, reports.
- [ ] Sanctum token auth + scopes.
- [ ] Rate limiting + throttling.
- [ ] API versioning (/api/v1).
- [ ] OpenAPI/Swagger docs.
- [ ] Webhook emitter for payment/booking events with retry.
- [ ] API key management in admin.

## Definition of Done
Third-party can authenticate and call documented endpoints.
